# Birth Time Rectification Verses from Brihat Parashar Hora Shastra

## Chapter: Lagna Adhyaya (लग्नाध्याय)

### Verse 1: Gulikadi Kala Jnana (गुलिकादि काल ज्ञान)

**Sanskrit Verse:**
```
रविवारादिशन्यन्तं गुलिकादि निरूप्यते।
दिवसानष्टधा कृत्वा वारेशाद्‌मणयेत््रमात्‌। ।१।।
```

**Word-to-Word Translation:**
- रविवारादि (ravi-vārādi) = from Sunday
- शनि-अन्तं (śani-antaṁ) = until Saturn/Saturday
- गुलिकादि (gulikādi) = Gulika etc.
- निरूप्यते (nirūpyate) = is determined
- दिवसान् (divasān) = day duration
- अष्टधा (aṣṭadhā) = into eight parts
- कृत्वा (kṛtvā) = dividing
- वारेशात् (vāreśāt) = from the lord of the day
- मणयेत् (maṇayet) = count/calculate
- क्रमात् (kramāt) = in sequence

**Complete Translation:**
From Sunday to Saturday, the determination of Gulika and similar upagrahas is made. By dividing the day duration into eight parts and counting from the lord of the day in sequence.

**Mathematical Formulas:**
1. Day Duration ÷ 8 = One Khanda (period)
2. Count from weekday lord to find Gulika period
3. For night birth: Night Duration ÷ 8, count from 5th weekday lord

**Example:**
If day duration is 32 ghatis 50 palas on Saturday:
- 32:50 ÷ 8 = 4:06:15 (one khanda)
- First khanda from Saturn = Gulika period

### Verse 2: Ashthamo Niresha (अष्टमो निरीश)

**Sanskrit Verse:**
```
अष्छ्मषो निरीशः स्याच्छन्यंशो गुरिकः स्मृतः
रात्रिरष्यष्टधा भक्त्वा वःरर््पञ्चनादतः।।२।।
```

**Word-to-Word Translation:**
- अष्टमः (aṣṭamaḥ) = eighth
- निरीशः (nirīśaḥ) = without lord
- स्यात् (syāt) = becomes
- शन्यंशः (śanyaṁśaḥ) = Saturn's portion
- गुलिकः (gulikaḥ) = Gulika
- स्मृतः (smṛtaḥ) = is remembered/known
- रात्रि (rātri) = night
- अष्टधा (aṣṭadhā) = eight-fold
- भक्त्वा (bhaktvā) = dividing
- वार-पञ्चनात् (vāra-pañcanāt) = from fifth weekday
- ततः (tataḥ) = then

**Complete Translation:**
The eighth portion has no lord, and Saturn's portion is known as Gulika. Similarly, for night births, divide the night duration into eight parts and count from the fifth weekday lord.

**Mathematical Formulas:**
1. 8th Khanda = No planetary lord
2. Saturn's Khanda = Gulika
3. For night: Count from 5th weekday lord

### Verse 3: Shanyamsho Gulikah (शन्यंशो गुलिकः)

```
गणयेदष्टमो खण्डो निष्पत्तिः परिकीितः।
```

- अष्टमः (aṣṭamaḥ) = eighth

**Complete Translation:**

**Mathematical Formulas:**

### Verse 5: Pranapada Sadhana (प्राणपद साधन) - Method 1

**Sanskrit Verse:**
```
घटी चतुर्गुणा कार्यां तिथ्याप्तैश्च पलैर्युताः
दिनकरेणापहतं शेषं प्राणपदं स्मृतम्‌।।५।।
```

**Word-to-Word Translation:**
- घटी (ghaṭī) = ghatis (time units)
- चतुर्गुणा (caturgaṇā) = multiplied by four
- कार्या (kāryā) = should be done
- तिथि-आप्तैः (tithi-āptaiḥ) = divided by tithi
- च (ca) = and
- पलैः (palaiḥ) = palas (time units)
- युताः (yutāḥ) = added
- दिनकरेण (dinakarena) = by twelve
- अपहतं (apahatam) = divided
- शेषं (śeṣaṁ) = remainder
- प्राणपदं (prāṇapadaṁ) = Pranapada
- स्मृतम् (smṛtam) = is remembered

**Complete Translation:**
The ghatis of Ishta Kala should be multiplied by four. The palas should be divided by 15 and the quotient added. When divided by 12, the remainder gives the rashi of Pranapada.

**Mathematical Formulas:**
1. Ishta Ghati × 4 = Base value
2. Ishta Pala ÷ 15 = Quotient
3. (Base value + Quotient) ÷ 12 = Pranapada rashi (remainder)
4. Remainder × 2 = Pranapada degrees

**Example:**
Ishta Kala = 32:50
- 32 × 4 = 128
- 50 ÷ 15 = 3, remainder 5
- 128 + 3 = 131
- 131 ÷ 12 = 10 remainder 11 (rashi)
- 5 × 2 = 10 degrees
- Medium Pranapada = 11 rashi 10 degrees

### Verse 6: Pranapada Amshakriya (प्राणपद अंशक्रिया)

**Sanskrit Verse:**
```
शेषात्यलान्तादिहवगुणी विधाय राश्यंशसूर्यक्षनियोजिताच।
तत्रापि तद्राशिच्छान्‌ क्रमेण लग्न साशपदेक्यता स्यात्‌। ।६।।
```

**Word-to-Word Translation:**
- शेषात् (śeṣāt) = from remainder
- अंश (aṁśa) = degrees
- द्विगुणी (dviguṇī) = doubled
- विधाय (vidhāya) = making
- राशि-अंश (rāśi-aṁśa) = sign and degrees
- सूर्य (sūrya) = Sun's
- नियोजिता (niyojitā) = added
- च (ca) = and
- तत्र-अपि (tatra-api) = therein also
- तद्-राशि (tad-rāśi) = that sign
- च (ca) = and
- क्रमेण (krameṇa) = in sequence
- लग्न (lagna) = ascendant
- अंश-पद (aṁśa-pada) = degree position
- ऐक्यता (aikyatā) = equality/unity
- स्यात् (syāt) = should be

**Complete Translation:**
The remainder doubled gives the degrees. These rashi-amshas, when added to the Sun's position according to its rashi nature (chara/sthira/dvisvabhava), give the sphuta Pranapada. The lagna degrees and Pranapada degrees should be equal.

**Mathematical Formulas:**
1. If Sun in Chara rashi: Add Sun's rashi-amsha to Pranapada
2. If Sun in Sthira rashi: Add 9th rashi-amsha from Sun
3. If Sun in Dvisvabhava rashi: Add 5th rashi-amsha from Sun
4. Lagna degrees = Pranapada degrees (for verification)

### Verse 7: Pranapada Sadhana - Method 2 (प्राणपद साधन - द्वितीय विधि)

**Sanskrit Verse:**
```
अथच--स्वेष्टकालं पलीकृत्य तिथ्याप्तं भादिकं च यत्‌।।७।।
चरागद्विभके भानौ योज्यं स्वे नवमे सुते।।७।।
स्फुट प्राणपदं तस्मात्‌ पूर्ववच्छोधयेत्तनुम्‌।।७।।
```

**Word-to-Word Translation:**
- अथ-च (atha-ca) = moreover
- स्व-इष्टकालं (sva-iṣṭakālaṁ) = one's Ishta Kala
- पलीकृत्य (palīkṛtya) = converting to palas
- तिथि-आप्तं (tithi-āptaṁ) = divided by 15
- भागदिकं (bhāgadikaṁ) = degrees etc.
- च (ca) = and
- यत् (yat) = which
- चर-आदि (cara-ādi) = movable etc.
- द्विभके (dvibhake) = dual-natured
- भानौ (bhānau) = when Sun is
- योज्यं (yojyaṁ) = should be added
- स्वे (sve) = own
- नवमे (navame) = ninth
- सुते (sute) = fifth
- स्फुट (sphuṭa) = accurate
- प्राणपदं (prāṇapadaṁ) = Pranapada
- तस्मात् (tasmāt) = from that
- पूर्ववत् (pūrvavat) = as before
- शोधयेत् (śodhayet) = should rectify
- तनुम् (tanum) = ascendant

**Complete Translation:**
Convert Ishta Kala to palas and divide by 15 to get rashi and degrees. When Sun is in chara rashi, add Sun's position; in sthira, add 9th position; in dvisvabhava, add 5th position to get sphuta Pranapada. From this, rectify the ascendant as before.

**Mathematical Formulas:**
1. Ishta Kala (in palas) ÷ 15 = rashi and degrees
2. Add based on Sun's rashi nature:
   - Chara: Sun's rashi-amsha
   - Sthira: Sun's rashi + 8 rashis
   - Dvisvabhava: Sun's rashi + 4 rashis
3. Result = Sphuta Pranapada

**Example:**
Ishta Kala = 32:28:30 = 1949:30 palas
- 1949:30 ÷ 15 = 129 rashi, remainder 13:30
- 13:30 × 2 = 27 degrees
- Sun in Gemini (chara) at 3r 18d
- 129r 27d + 3r 18d = 133r 15d = 1r 15d

### Verse 8: Ashuddha Lagna Lakshana (अशुद्ध लग्न लक्षण)

**Sanskrit Verse:**
```
विना प्राणपदाच्छुद्धो गुलिकाद्वा निशाकराद्‌।
तदशद्धं विजानीयात्स्थावराणां तदेव दहि।।८।।
```

**Word-to-Word Translation:**
- विना (vinā) = without
- प्राणपदात् (prāṇapadāt) = from Pranapada
- शुद्धः (śuddhaḥ) = purified/rectified
- गुलिकात् (gulikāt) = from Gulika
- वा (vā) = or
- निशाकरात् (niśākarāt) = from Moon
- तत् (tat) = that
- अशुद्धं (aśuddhaṁ) = impure/incorrect
- विजानीयात् (vijānīyāt) = should know
- स्थावराणां (sthāvarāṇāṁ) = of immovable things
- तत् (tat) = that
- एव (eva) = indeed
- हि (hi) = certainly

**Complete Translation:**
A birth lagna that has not been rectified through Pranapada, Gulika, or Moon should be known as impure/incorrect, and it indicates the birth of immovable things (non-living entities or inanimate objects).

**Mathematical Formulas:**
1. Lagna degrees should equal Pranapada degrees
2. Alternative: Verify with Gulika lagna
3. Alternative: Verify with Moon position
4. If none match → Incorrect birth time (or non-human birth)

### Verse 9: Lagna Shodhana Vidhi (लग्न शोधन विधि)

**Sanskrit Verse:**
```
दयोहीनबलेऽप्येवं गुलिकात्परिचिन्तयेत्‌।
तस्मात्तत्सप्तमस्थात्तदं शाच्च कलत्रतः।।९।।
```

**Word-to-Word Translation:**
- द्वयोः (dvayoḥ) = of both
- हीन-बले (hīna-bale) = when weak
- अपि (api) = even
- एवं (evaṁ) = thus
- गुलिकात् (gulikāt) = from Gulika
- परिचिन्तयेत् (paricintayet) = should consider
- तस्मात् (tasmāt) = from that
- तत्-सप्तम (tat-saptama) = its seventh
- स्थात् (sthāt) = from position
- तत्-अंशात् (tad-aṁśāt) = from its degrees
- च (ca) = and
- कलत्रतः (kalatrataḥ) = from seventh house

**Complete Translation:**
Even when both (Pranapada and lagna) are weak, one should consider rectification through Gulika, or from the seventh position from it, or from its degrees, or from the seventh house.

**Mathematical Formulas:**
1. If Pranapada weak: Use Gulika lagna
2. Alternative: Use 7th from Gulika
3. Alternative: Use degrees of Gulika
4. Alternative: Use 7th house from lagna

### Verse 10: Trikona Niyama (त्रिकोण नियम)

**Sanskrit Verse:**
```
तथैव तत्त्निकोणे वा जन्मलग्नं विनिर्दिशेत्‌
मनुष्याणां पशूनां य द्दितीये दशमे रिपौ।।१०।।
```

**Word-to-Word Translation:**
- तथा-एव (tathā-eva) = similarly
- तत्-त्रिकोणे (tat-trikoṇe) = in its trine
- वा (vā) = or
- जन्म-लग्नं (janma-lagnaṁ) = birth ascendant
- विनिर्दिशेत् (vinirdiśet) = should be determined
- मनुष्याणां (manuṣyāṇāṁ) = of humans
- पशूनां (paśūnāṁ) = of animals
- य (ya) = which
- द्वितीये (dvitīye) = in second
- दशमे (daśame) = in tenth
- रिपौ (ripau) = in sixth/enemy

**Complete Translation:**
The birth ascendant should be determined in the trine (1st, 5th, 9th houses) from Pranapada for humans. For animals, it is in the 2nd, 6th, or 10th houses from Pranapada.

**Mathematical Formulas:**
1. For human births: Lagna in 1, 5, or 9 from Pranapada rashi
2. For animal births: Lagna in 2, 6, or 10 from Pranapada rashi
3. For birds: Lagna in 3 or 7 from Pranapada
4. For reptiles/aquatics: Other positions

**Example:**
If Pranapada = 11 rashi (Aquarius)
- Human lagna: 11, 3, or 7 rashi (Aquarius, Aries, or Leo)
- Animal lagna: 12, 4, or 8 rashi

### Verse 14: Nisheka Lagna Ganana (निषेक लग्न गणना)

**Sanskrit Verse:**
```
यस्मिन्‌ भावे स्थितो कोणस्तस्य मान्देर्यदन्तरम्‌।
लग्नभाग्यन्तरं योज्यं यच्च राश्यादि जायते।।१४।।
```

**Word-to-Word Translation:**
- यस्मिन् (yasmin) = in which
- भावे (bhāve) = house
- स्थितः (sthitaḥ) = situated
- कोणः (koṇaḥ) = Saturn
- तस्य (tasya) = of that
- मान्देः (māndeh) = Gulika's
- यत् (yat) = which
- अन्तरम् (antaram) = difference
- लग्न-भाग्य (lagna-bhāgya) = lagna and ninth house
- अन्तरं (antaraṁ) = difference
- योज्यं (yojyaṁ) = should be added
- यत् (yat) = which
- च (ca) = and
- राशि-आदि (rāśi-ādi) = sign etc.
- जायते (jāyate) = results

**Complete Translation:**
Find the house in which Saturn is placed, calculate the difference between that house and Gulika lagna. Add to this the difference between lagna and the ninth house. The resulting rashi etc. indicates the period before birth.

**Mathematical Formulas:**
1. Find Saturn's bhava (house)
2. Saturn's bhava - Gulika lagna = Difference A
3. Lagna - 9th house = Difference B
4. Difference A + Difference B = Period before birth (in months)

**Example:**
If Saturn at 7r 11d 58s, Gulika at 4r 9d 26s
- Difference = 3r 2d 21s 58p
- Lagna 9r 15d 23s, 9th house 5r 24d 41s
- Difference = 4r 20d 51s 26p
- Total = 7r 23d 22s 24p = 7 months 22 days before birth

### Verse 15: Chandrabhuktamsha Yoga (चन्द्रभुक्तांश योग)

**Sanskrit Verse:**
```
मासादिस्तन्मितं ज्ञेयं जन्मतः प्राक्‌ निषेकजम्‌।
यदादृश्यदलेऽङ्गेशस्तदेन्दोभुक्तभागयुक्‌।।१५।।
```

**Word-to-Word Translation:**
- मास-आदिः (māsa-ādiḥ) = months etc.
- तन्मितं (tanmitaṁ) = that measure
- ज्ञेयं (jñeyaṁ) = should be known
- जन्मतः (janmataḥ) = from birth
- प्राक् (prāk) = before
- निषेक-जं (niṣeka-jaṁ) = of conception
- यदा (yadā) = when
- अदृश्य-दले (adṛśya-dale) = invisible half
- अङ्ग-ईशः (aṅga-īśaḥ) = lagna lord
- तदा (tadā) = then
- इन्दोः (indoḥ) = Moon's
- भुक्त-भाग (bhukta-bhāga) = traversed degrees
- युक् (yuk) = added

**Complete Translation:**
This measure in months etc. should be known as the period before birth for conception. When the lagna lord is in the invisible half (7th to 12th houses from lagna), add the degrees traversed by the Moon to get the accurate period.

**Mathematical Formulas:**
1. Calculate period from verse 14 formula
2. If lagna lord in 7-12 houses from lagna:
   - Add Moon's traversed degrees in its rashi
3. Final result = Conception time before birth

### Verse 16: Garbhastha Phala (गर्भस्थ फल)

**Sanskrit Verse:**
```

```

**Word-to-Word Translation:**
- तत्-काले (tat-kāle) = at that time
- साधयेत् (sādhayet) = should calculate
- लग्नं (lagnaṁ) = ascendant
- शोधयेत् (śodhayet) = should rectify
- पूर्ववत् (pūrvavat) = as before
- तनुम् (tanum) = body/ascendant
- तस्मात् (tasmāt) = from that
- लग्नात् (lagnāt) = from ascendant
- फलं (phalaṁ) = results
- वाच्यं (vācyaṁ) = should be predicted
- गर्भ-स्थस्य (garbha-sthasya) = of fetus
- विशेषतः (viśeṣataḥ) = especially

**Complete Translation:**
At that calculated conception time, compute the lagna and rectify it as before. From that conception lagna, predictions should be made, especially regarding the fetus.

**Mathematical Formulas:**
1. Go back calculated period from birth
2. Calculate lagna for that moment = Nisheka lagna
3. Use Pranapada method to verify/rectify
4. Predict fetal conditions, parent's conditions from Nisheka chart

### Verse 18: Bhava Lagna Sadhana (भाव लग्न साधन)

**Sanskrit Verse:**
```
सूर्योदयात्समारभ्य घटीपञ्च प्रमाणतः।
जन्मेष्टकालपर्यन्तं गणनीयं प्रयत्नतः।।१८।।
```

**Word-to-Word Translation:**
- सूर्य-उदयात् (sūrya-udayāt) = from sunrise
- सम-आरभ्य (sama-ārabhya) = beginning
- घटी-पञ्च (ghaṭī-pañca) = five ghatis
- प्रमाणतः (pramāṇataḥ) = as measure
- जन्म-इष्ट-काल (janma-iṣṭa-kāla) = birth Ishta Kala
- पर्यन्तं (paryantaṁ) = until
- गणनीयं (gaṇanīyaṁ) = should be counted
- प्रयत्नतः (prayatnataḥ) = carefully

**Complete Translation:**
From sunrise, considering five ghatis as the measure for one lagna, count carefully until the birth Ishta Kala. This gives the Bhava Lagna (also called Ghati Lagna).

**Mathematical Formulas:**
1. Time from sunrise to birth (in ghatis) ÷ 5 = Number of lagnas elapsed
2. If odd lagna: Count from Sun's rashi
3. If even lagna: Count from Janma lagna
4. Result = Bhava Lagna

**Example:**
Ishta Kala = 32:28:30, Sun at 3r 18d 26s
- 32:28:30 ÷ 5 = 6:29:42 (lagna count)
- Add to Sun: 3r 18d 26s + 6r 29d 42s = 10r 28d 8s
- Bhava Lagna = 10r 28d 8s

### Verse 20-21: Hora Lagna Vidhi (होरा लग्न विधि)

**Sanskrit Verse:**
```
सार्धद्विष्ठटिक्छा विप्र कालादिति विलग्नभात्‌।
श्रयन्ति लम्नं तत्रामर होरालग्नं द्विजोत्तम।।२०।।

समलग्ने यदा जन्म गणयेज्जन्मभादद्धिज।।२१।।
```

**Word-to-Word Translation:**
- सार्ध-द्वि-घटिका (sārdha-dvi-ghaṭikā) = two and half ghatis
- विप्र (vipra) = O brahmin

- इति (iti) = thus
- लग्न-भात् (lagna-bhāt) = lagna results

- होरा-लग्नं (horā-lagnaṁ) = Hora lagna

- जन्म (janma) = birth
- विषम-राशिषु (viṣama-rāśiṣu) = in odd signs

- जन्म-भावात् (janma-bhāvāt) = from birth house

**Complete Translation:**
O best of brahmins! Two and half ghatis constitute one Hora lagna. When birth is in odd rashi, count from Sun's rashi; when in even rashi, count from birth lagna.

**Mathematical Formulas:**

### Verse 22-23: Ghati Lagna Sadhana (घटी लग्न साधन)

**Sanskrit Verse:**
```
घटीलग्नं प्रवक्ष्यामि शृणु त्वं द्विजसत्तम।
सूर्योदयात्समारभ्य जन्मकालावधि क्रमात्‌।।२२।।

```

**Word-to-Word Translation:**
- घटी-लग्नं (ghaṭī-lagnaṁ) = Ghati lagna
- प्रवक्ष्यामि (pravakṣyāmi) = I shall explain
- शृणु (śṛṇu) = listen
- त्वं (tvaṁ) = you
- द्विज-सत्तम (dvija-sattama) = O best of twice-born
- सूर्य-उदयात् (sūrya-udayāt) = from sunrise

- जन्म-काल (janma-kāla) = birth time
- अवधि (avadhi) = until

**Complete Translation:**

**Mathematical Formulas:**

### Verse 26-28: Varnada Lagna Sadhana (वर्णद लग्न साधन)

**Sanskrit Verse:**
```
जन्महोराख्यलग्नक्षसंख्या ग्राह्या पृथक्‌ पृथक्‌।
ओजे लग्ने त्वेकयुग्मे चक्रशुद्धैकसंयुता।।२६।।
युग्मौजसाम्ये संयोज्य वियोज्यान्योनमन्यथा।
मेषादितः क्रमादोजे मीनादेरुत्क्रमात्समे।।२७।।
एवं यल्लग्नमायाति वर्णदं तत्‌ प्रकोत्तितम्‌।
एवं द्ादशभावानां वणंदं लग्नमानयेत्‌।।२८।।
```

**Word-to-Word Translation:**
- जन्म-होरा-आख्य (janma-horā-ākhya) = birth and hora
- लग्न-राशि (lagna-rāśi) = ascendant signs
- संख्या (saṁkhyā) = numbers
- ग्राह्या (grāhyā) = should be taken
- पृथक्-पृथक् (pṛthak-pṛthak) = separately
- ओजे (oje) = odd
- लग्ने (lagne) = in ascendant
- एक-युग्मे (eka-yugme) = one even
- चक्र-शुद्ध (cakra-śuddha) = from 12 subtracted
- एक-संयुता (eka-saṁyutā) = added to one
- युग्म-ओज (yugma-oja) = even-odd
- साम्ये (sāmye) = when equal
- संयोज्य (saṁyojya) = should be added
- वियोज्य (viyojya) = should be subtracted
- अन्योन्यम् (anyonyam) = mutually
- अन्यथा (anyathā) = otherwise
- मेष-आदितः (meṣa-āditaḥ) = from Aries
- क्रमात् (kramāt) = in order
- ओजे (oje) = in odd
- मीन-आदेः (mīna-ādeḥ) = from Pisces
- उत्क्रमात् (utkramāt) = in reverse
- समे (same) = in even
- एवं (evaṁ) = thus
- यत् (yat) = which
- लग्नम् (lagnam) = ascendant
- आयाति (āyāti) = comes
- वर्णदं (varṇadaṁ) = Varnada
- तत् (tat) = that
- प्रकीर्तितम् (prakīrtitam) = is proclaimed
- द्वादश-भावानां (dvādaśa-bhāvānāṁ) = of twelve houses
- लग्नम् (lagnam) = ascendant
- आनयेत् (ānayet) = should bring

**Complete Translation:**
Take the numbers of birth lagna and hora lagna separately. If both are odd or both are even, add them (if sum is even, subtract from 12 first). If one is odd and one is even, subtract the smaller from 12 and then find difference. Count from Aries if odd, from Pisces in reverse if even. The resulting lagna is called Varnada. Similarly, calculate Varnada for all twelve houses.

**Mathematical Formulas:**
**Case 1:** Both Janma lagna and Hora lagna are odd:
- Add both
- If sum is even, subtract from 12
- Count from Aries = Varnada

**Case 2:** Both are even:
- Subtract from 12 separately
- Add results
- If sum is even, subtract from 12 again
- Count from Aries = Varnada

**Case 3:** One odd, one even:
- Subtract even from 12
- Subtract result from odd
- If result is even, subtract from 12
- Count from Aries = Varnada

**Example:**
Janma lagna = 9 (Sagittarius - odd)
Hora lagna = 4 (Cancer - even)
- Subtract 4 from 12 = 8
- 9 - 8 = 1 (Aries)
- But since we need odd, and 1 is already odd
- Varnada = 2r (Taurus) after proper calculation
