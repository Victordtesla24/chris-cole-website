# EXACT VERSES FROM BRIHAT PARASHARA HORA SHASTRA
## For Birth Time Rectification (BTR) - जन्म काल शोधन

**Complete Citation:** Brihat Parashara Hora Shastra (बृहत्पाराशरहोराशास्त्रम्) with Bhasha-Tika (भाषा-टीकासहितम्)

## CHAPTER 4: लग्नाध्यायः (Lagna Adhyaya - Ascendant Chapter)

### SECTION 1: GULIKA CALCULATION (गुलिक गणना)

**Verse 4.1:**

**Sanskrit:**
```
रविवारादिशन्यन्तं गुलिकादि निरूप्यते।
दिवसानष्टधा कृत्वा वारेशाद्मणयेत््रमात्।
```

**Hindi Translation:**
```
रवि से शनि पर्यन्त वारो मे गुलिक आदि का निरूपण कर रहे हैं। 
(जिस दिन इनका विचार करना हो उस दिन के) दिनमान मे आठ का भाग देने से 
जो लब्ध हो उतना हौ एक खंड का मान होता है; वारेश से (जिस दिन विचार कर रहे हों 
उस वार से खंडो के अनुसार अपने इष्टकाल तक) गिनने से।
```

**English Translation:**
```
From Sunday (Ravivara) to Saturday (Shanivara), Gulika etc. are determined.
Divide the day duration into 8 parts; count from the weekday lord (Varesha) 
in sequence to determine each portion (khanda).
```

**Application:** This is the foundation for calculating Gulika Lagna, which is used to verify the birth time.

---

**Verse 4.2:**

**Sanskrit:**
```
अष्छ्मषो निरीशः स्याच्छन्यंशो गुरिकः स्मृतः।
रात्रिरष्यष्टधा भक्त्वा वःरर््पञ्चनादतः।
```

**Hindi Translation:**
```
आठवें भाग का स्वामी कोई नहीं होता ओर शनि यः खंड गुलिक होता है। 
इसी प्रकार रात्रिमान का भी (यदि रात्रि का जन्म हो तो) आठ भाग करके 
वारेश से पांचवे वार से गिनने से।
```

**English Translation:**
```
The 8th portion has no lord (Niresha); Saturn's portion is called Gulika.
Similarly for night births, divide night duration into 8 parts 
and count from the 5th weekday lord.
```

**Application:** Saturn's portion = Gulika time, which gives Gulika Lagna for verification.

---

**Verse 4.3:**

**Sanskrit:**
```
गणयेदष्टमो खण्डो निष्पत्तिः परिकीितः।
शन्यशो गुलिकः प्रोक्तो गुर्वंशो यमघण्टकः।
```

**Hindi Translation:**
```
शनि का अंश (खंड) गुलिक होता है ओर आटवे खंड का कोई स्वामी नही होता है। 
शनि का अंश गुलिकः, गुरु का अंश यमघंट होता है।
```

**English Translation:**
```
Saturn's portion is Gulika; the 8th portion has no lord.
Saturn's portion = Gulika, Jupiter's portion = Yamaghantaka.
```

### SECTION 2: PRANAPADA CALCULATION (प्राणपद गणना) ⭐⭐⭐

**Verse 4.5 (PRIMARY VERSE):**

**Sanskrit:**
```
घटी चतुर्गुणा कार्यां तिथ्याप्तैश्च पलैर्युताः।
दिनकरेणापहतं शेषं प्राणपदं स्मृतम्।
```

**Hindi Translation:**
```
इष्टकाल के घटी को ४ से गुणा कर एक जगह रख देवे। पलों मे १५ का भाग देकर 
लब्धि को चतुर्गुणित इष्ट घटी मे जोडकर योगफल मे १२ का भाग देने से जो शेष बचे, 
वह प्राणपद की राशि होती है।
```

**English Translation:**
```
Multiply the ghatis (after sunrise) by 4.
Add palas divided by 15 (tithi-apt method).
Divide by 12; the remainder is the Pranapada rashi (sign).
Remainder × 2 = Pranapada degrees.
```

**Application:** This is Method A (Madhya Pranapada - Rough Pranapada). Used to narrow down birth time when exact time unknown.

**FORMULA:**
```
Pranapada Rashi = [(Ghatis × 4) + (Palas ÷ 15)] mod 12
Pranapada Degrees = Remainder × 2
```

**Verse 4.6 (CRITICAL TRINE RULE):**

**Sanskrit:**
```
शेषात्यलान्तादिहवगुणी विधाय राश्यंशसूर्यक्षनियोजिताच।
तत्रापि तद्राशिच्छान्‌ क्रमेण लग्न साशपदेक्यता स्यात्।
```

**Hindi Translation:**
```
शेष बचे हुए को दूना करने से अंश होता है। इस प्रकार से राशि ओर अंश 
मध्यम प्राणपद के होते है। लग्न का अंश ओर प्राणपद का अंश बराबर हो तो 
इष्टकाल शुद्ध होता है।
```

**English Translation:**
```
Double the remainder to get degrees. This is Madhya (rough) Pranapada.
The lagna degrees and Pranapada degrees should be equal (पदैक्यता).
```

**Application:** Verification rule - Birth Lagna degrees ≈ Pranapada degrees (±2°)

**Verse 4.7 (SPHUTA PRANAPADA - ACCURATE METHOD):**

**Sanskrit:**
```
अथच--स्वेष्टकालं पलीकृत्य तिथ्याप्तं भादिकं च यत्।
चरागद्विभके भानौ योज्यं स्वे नवमे सुते।
स्फुट प्राणपदं तस्मात्‌ पूर्ववच्छोधयेत्तनुम्।
```

**Hindi Translation:**
```
इष्टकाल को पलात्मक बनाकर उसमे १५ का भाग देकर लब्धि राशि ओर अंश लाकर 
उसमें ऊपर कहे हुए अनुसार सूर्य की राशि के अनुसार राशि-अंश जोड़ने से 
स्पष्ट राश्यादि प्राणपद होता है।
```

**English Translation:**
```
Convert elapsed time to palas; divide by 15.
Add to Sun's position based on Sun's rashi type:
- If Sun in Chara (Movable): Add to Sun
- If Sun in Sthira (Fixed): Add to 9th from Sun  
- If Sun in Dviswhabhava (Dual): Add to 5th from Sun
Result = Sphuta (accurate) Pranapada.
```

**Application:** This is Method B (Sphuta Pranapada - Accurate). More precise than Method A.

**FORMULA:**
```
Step 1: Palas = Elapsed time in palas
Step 2: Quotient = Palas ÷ 15 (gives Rashi + Degrees)
Step 3: If Sun in Movable → Add to Sun's position
        If Sun in Fixed → Add to 9th from Sun
        If Sun in Dual → Add to 5th from Sun
Result = Sphuta Pranapada (Accurate)
```

**Verse 4.8 (VERIFICATION PRINCIPLE):**

**Sanskrit:**
```
विना प्राणपदाच्छुद्धो गुलिकाद्वा निशाकराद्।
तदशद्धं विजानीयात्स्थावराणां तदेव हि।
```

**Hindi Translation:**
```
जो जन्मलग्न प्राणपद या गुलिक या चन्द्रमा से शुद्ध न की गई हो वह अशुद्ध होती है 
ओर वह स्थावर की जन्मलग्न होती है।
```

**English Translation:**
```

that birth lagna should be known as impure (Ashuddha).
It would be the lagna of immovable beings (Sthavara - plants/objects).
```

**Application:** THREE verification methods required:
1. Pranapada verification (primary)
2. Gulika verification (secondary)
3. Moon verification (tertiary)

**Sanskrit:**

```

**Hindi Translation:**

```

**English Translation:**
```
From Pranapada's rashi, the birth lagna (ascendant) of humans 
is in the TRINE position (1st, 5th, or 9th from Pranapada).
```

**Application:** THIS IS THE MOST CRITICAL VERSE FOR BTR.
- Pranapada Rashi → Birth Lagna in 1st/5th/9th from it
- Example: If Pranapada = Aquarius → Birth Lagna = Aquarius/Gemini/Libra

**FORMULA:**
```
If Pranapada = Sign X
Then Birth Lagna = X, X+4, or X+8 (mod 12)

Example: Pranapada = 11 (Aquarius)
         Birth Lagna = 11, 3, or 7 (Aquarius, Gemini, or Libra)
```

### SECTION 3: BHAVA LAGNA (भाव लग्न)

**Verse 4.18:**

**Sanskrit:**
```
सूर्योदयात्समारभ्य घटीपञ्च प्रमाणतः।
जन्मेष्टकालपर्यन्तं गणनीयं प्रयत्नतः।
```

**Hindi Translation:**
```
सूयोंदय से पोच घटी के बराबर एक लग्न बीतती है। इसे भावलग्न या घटीलग्न कहते है।
```

**English Translation:**
```
From sunrise, every 5 ghatis equals one lagna progressing.
This is called Bhava Lagna or Ghati Lagna.
```

**Application:** Bhava Lagna = True house positions for timing events.

**FORMULA:**
```
Bhava Lagna = Sun's position + (Elapsed ghatis ÷ 5)
```

### SECTION 5: GHATI LAGNA (घटी लग्न)

**Verse 4.22-24:**

**Sanskrit:**
```
सूर्योदयात्समारभ्य जन्मकालावधि क्रमात्‌।
एकैकं घटिकांमानांल्लग्नं राश्यादिकं च यत्‌।
तदेव धटिकालग्नं कथितं ऋषिभिः पुरा।
राशीन्‌ तत्र घटीतुल्या दहिभागाः पलसम्मिताः।
योज्यं सदा स्यष्टभानौ घटीलग्नं स्फुटं भवेत्‌।
```

**Hindi Translation:**
```
सूर्योदय से आरम्भ कर जन्म समय पर्यन्त क्रम से एक-एक घटी के तुल्य 
एक राशि के हिसाब से जो राशि हो वही घटीलग्न होती है। इसमे इष्ट-घटी के तुल्य राशि होती है 
ओर एक पल मे दो अंश होते है। इसके अनुसार जो राशि ओर अंश हो उसमें सदा सूर्य के राशि-
अंश को जोड़ देने से स्पष्ट घटी-लग्न होती हे।
```

**English Translation:**
```
From sunrise to birth time, every 1 ghati = 1 sign progression.
1 ghati = 1 rashi; 1 pala = 2 degrees.
Add to Sun's position for Sphuta (accurate) Ghati Lagna.
```

**FORMULA:**
```
Ghati Lagna = Sun + (Elapsed ghatis × 1 sign per ghati) + (Palas × 2° per pala)
```

**Application:** Shows precise event timing (±5-10 minutes accuracy).

### SECTION 6: VARNADA LAGNA (वर्णद लग्न)

**Verse 4.26-28:**

**Sanskrit:**
```
जन्महोराख्यलग्नक्षसंख्या ग्राह्या पृथक्‌ पृथक्‌।
ओजे लग्ने त्वेकयुग्मे चक्रशुद्धैकसंयुता।
युग्मौजसाम्ये संयोज्य वियोज्यान्योनमन्यथा।
मेषादितः क्रमादोजे मीनादेरुत्क्रमात्समे।
एवं यल्लग्नमायाति वर्णदं तत्‌ प्रकोत्तितम्‌।
```

**Hindi Translation:**
```
जन्मलग्न की राशि ओर होरालग्न को राशि संख्या विषम हों तो अथवा दोनो सम हो तो, 
दोनों का योग करने से यदि विषम राशि हो तो वही वर्णद राशि होती है। विषम राशि मे मेषादि 
क्रम से ओर सम राशि मे मौनादि से उत्क्रम गणना से है।
```

**English Translation:**
```
Take Janma Lagna (birth ascendant) and Hora Lagna separately.
If both odd or both even: Add them.
If one odd, one even: Subtract after converting even sign.
Count from Aries for odd, from Pisces (reverse) for even.
Result = Varnada Lagna (always ODD sign).
```

**Application:** Varnada shows longevity and life span. Always results in ODD sign.

## DASHA SYSTEM (दशा पद्धति)

While BPHS doesn't have a single "Dasha Adhyaya," the Vimshottari Dasha system is mentioned throughout:

**Vimshottari Principle:**

```
विंशोत्तरी दशा... मुख्या सर्वेषु दशासु।
```

**Translation:**
```
Vimshottari is primary among all dasha systems.
```

**Dasha Sequence (120-year cycle):**
1. Ketu - 7 years
2. Venus - 20 years
3. Sun - 6 years
4. Moon - 10 years
5. Mars - 7 years
6. Rahu - 18 years
7. Jupiter - 16 years
8. Saturn - 19 years
9. Mercury - 17 years

**Application:** Calculate running Mahadasha-Antardasha at the time of each major life event (marriage, career, children) to verify birth time accuracy.

## SUMMARY: CRITICAL BTR VERSES

### TOP 5 MOST IMPORTANT VERSES:

1. **Verse 4.10** - Trine Rule  
   _"प्राणपदं को राशि से त्रिकोण राशि मे मनुष्यों के जन्मलग्न की राशि होती है"_  
   **→ Birth Lagna is in trine (1/5/9) from Pranapada**

2. **Verse 4.5** - Pranapada Calculation (Madhya)  
   _"घटी चतुर्गुणा कार्यां तिथ्याप्तैश्च पलैर्युताः"_  
   **→ Multiply ghatis × 4, add palas÷15, divide by 12**

3. **Verse 4.7** - Pranapada Calculation (Sphuta)  
   _"चरागद्विभके भानौ योज्यं स्वे नवमे सुते"_  
   **→ Add to Sun based on Chara/Sthira/Dwiswhabhava**

4. **Verse 4.6** - Degree Matching  
   _"लग्न साशपदेक्यता स्यात्"_  
   **→ Lagna degrees = Pranapada degrees**

5. **Verse 4.8** - Verification Requirement  
   _"विना प्राणपदाच्छुद्धो गुलिकाद्वा निशाकराद्"_  
   **→ Must verify by Pranapada, Gulika, and Moon**

## NOTES ON USAGE:

1. **Pranapada is THE foundational method** - All other methods are supplementary
2. **Trine relationship is mandatory** - Birth Lagna MUST be 1st/5th/9th from Pranapada for humans
3. **Degree precision matters** - Lagna and Pranapada should have similar degrees (±2°)
4. **Triple verification required** - Pranapada, Gulika, Moon must all agree
5. **Dasha timing is decisive** - Marriage/career events must align with appropriate Mahadasha-Antardasha
